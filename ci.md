# 持续构建 Continuous integration

## 什么是CI

持续集成指的是，频繁地（一天多次）将代码集成到主干

## CI是如何工作的

1. 开发者编写代码，提交commit到仓库
2. CI服务检测到更改
3. CI编译系统，
4.
